REPO-Tool - Usage:

To use the repo-tool, you must pull the repository first. To do so, enter the following command in your CygWin/Linux shell: git clone https://github.com/GymWenFLL/tpp_repo

To use the tool enter the following: repo <update>

repo pulls the source-code only. repo update pulls updates from the source code.
